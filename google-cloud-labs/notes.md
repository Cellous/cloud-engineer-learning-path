## Storage Bucket

A bucket is a logical container for objects stored in Google Cloud Storage.

Characteristics:
- Globally unique name
- Stores objects (files)
- Highly durable
- Scalable
- Used for backups, static websites, logs, media, and datasets

---

If someone asks:

"What is a Cloud Storage bucket?"

You can answer:

"A Cloud Storage bucket is a globally unique container used to store objects such as images, videos, backups, logs, and application data. It provides scalable, durable object storage and is a foundational service in Google Cloud."
