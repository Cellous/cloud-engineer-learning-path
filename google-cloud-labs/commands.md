---

If an interviewer asks:

"What is Cloud Shell?"

A strong answer is:

"Cloud Shell is a browser-based Linux environment hosted by Google Cloud. It includes the Google Cloud SDK, provides built-in authentication to the active project, offers a persistent 5 GB home directory, and runs on a temporary Compute Engine virtual machine for managing cloud resources from the command line."
