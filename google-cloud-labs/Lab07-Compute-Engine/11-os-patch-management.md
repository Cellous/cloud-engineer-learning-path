# OS Patch Management

Google Cloud includes centralized operating system patch management for Compute Engine.

---

## Benefits

- Keep systems up to date
- Reduce security vulnerabilities
- Automate updates
- Centralized management

---

## Two Main Components

### Patch Compliance Reporting

Provides

- Patch status
- Missing updates
- Recommendations

---

### Patch Deployment

Automates

- Operating system updates
- Software updates

Runs Patch Jobs on selected VM instances.

---

## Patch Management Tasks

- Create Patch Approvals
- Flexible Scheduling
- Recurring Updates
- One-Time Updates
- Pre-Patch Scripts
- Post-Patch Scripts
- Centralized Administration

---

## Supported Operating Systems

- Linux
- Windows

---

## Why It Matters

Long-running VMs require regular updates to:

- Improve security
- Fix bugs
- Meet compliance requirements
