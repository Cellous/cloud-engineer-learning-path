# Actions Supported on a Terminated VM

When a VM is stopped (terminated), CPU and RAM billing stop.

However, charges continue for:

- Persistent Disks
- Reserved Static IP Addresses

---

## Supported Actions

✔ Change Machine Type

✔ Move to another Network

✔ Add or Remove Disks

✔ Modify Auto Delete Settings

✔ Change Instance Tags

✔ Modify Metadata

✔ Add or Remove Static IP

✔ Change Availability Policy

---

## Not Supported

❌ Change Boot Image

A stopped VM cannot change its boot image.

To use another operating system image, create a new VM.
