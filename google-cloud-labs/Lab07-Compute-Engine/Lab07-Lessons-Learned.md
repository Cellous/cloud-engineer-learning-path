# Lessons Learned

- Compute Engine is Google's IaaS offering.

- A VM always requires an Internal IP address.

- External IP addresses are optional.

- Machine type affects:
  - CPU
  - RAM
  - Network throughput
  - Cost

- SSD offers higher IOPS.

- Local SSD is temporary.

- Google provides TPUs for Machine Learning.

- Compute Engine automatically generates equivalent gcloud commands.

- Compute Engine supports Linux and Windows operating systems.

- VM pricing depends on:
  - Region
  - Machine Type
  - Storage
  - Networking
