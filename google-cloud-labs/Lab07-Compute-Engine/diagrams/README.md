# Diagrams

This folder contains editable Draw.io diagrams and exported graphics.

## File Types

- `.drawio` — Editable source
- `.svg` — Vector graphic (recommended for GitHub)
- `.png` — Raster image for presentations and documents

## Naming Convention

- *-overview = simplified diagram
- main name = detailed reference diagram

Example

vm-lifecycle-overview.svg
vm-lifecycle.svg
