# ☁️ Cloud Storage Recognition Patterns

| Concept                         | ACE Meaning                 |
| ------------------------------- | --------------------------- |
| Bucket                          | Storage container           |
| Object storage                  | Binary/unstructured data    |
| Multi-region                    | Large geographic redundancy |
| Dual-region                     | Two-region redundancy       |
| Regional                        | Lowest latency in one area  |
| `gcloud storage buckets create` | Modern CLI command          |
| `gsutil`                        | Older/legacy tooling        |
| ACLs not needed                 | Use IAM instead             |
