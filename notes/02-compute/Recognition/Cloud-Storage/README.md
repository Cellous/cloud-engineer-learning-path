## Cloud Storage CLI Recognition

- gcloud storage buckets create
  -> Create storage bucket

- gcloud storage cp
  -> Upload/download objects

- gcloud storage ls
  -> List buckets/objects

- gsutil
  -> Older storage CLI tooling

Recognition patterns:
- multi-region -> large geographic redundancy
- regional -> lowest latency
- dual-region -> geo redundancy across two regions
- immutable objects -> Cloud Storage
