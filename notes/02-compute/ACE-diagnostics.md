# Major ACE Concepts

| Technology     | Purpose                               |
| -------------- | ------------------------------------- |
| `gcloud`       | Manage Google Cloud resources         |
| `kubectl`      | Manage Kubernetes clusters/workloads  |
| Terraform      | Declarative infrastructure automation |
| GKE            | Managed Kubernetes clusters           |
| Cloud Run      | Serverless containers                 |
| Compute Engine | Virtual machines                      |

---

# Recognition patterns

- More flexibility/customization → Standard GKE
- Less operational overhead → Autopilot
- High availability → Regional
- Internal-only access → Private cluster
