# Cloud Run vs App Engine vs Cloud Functions

| Service | Best For | Key Trait |
|---|---|---|
| Cloud Run | Containers | Scale-to-zero serverless containers |
| Cloud Functions | Event-driven code | Small snippets/functions |
| App Engine Standard | Simple apps | Fast scaling but runtime restrictions |
| App Engine Flexible | Custom runtimes | More VM-based overhead |
| GKE | Full orchestration | Kubernetes control |
