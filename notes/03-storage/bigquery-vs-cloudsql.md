# BigQuery vs Cloud SQL

| Feature | BigQuery | Cloud SQL |
|---|---|---|
| Type | Data Warehouse | Relational Database |
| Workload | Analytical | Transactional |
| SQL Support | Yes | Yes |
| Scaling | Massive analytics | Application backend |
| Use Cases | Dashboards, BI | Ecommerce, login systems |

## Key Difference
BigQuery is OLAP.
Cloud SQL is OLTP.
