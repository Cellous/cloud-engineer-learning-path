# Google Cloud Spanner

## What It Is
Globally distributed relational database service.

## Recognition Patterns
- Globally scalable relational database
- Horizontal scaling
- Strong consistency
- Financial transactions
- Multi-region applications

## Best Use Cases
- Banking systems
- Inventory systems
- Global ecommerce
- Multi-region transactional apps

## Compared to Cloud SQL
| Cloud SQL | Spanner |
|---|---|
| Vertical scaling | Horizontal scaling |
| Regional | Global |
| Traditional relational | Distributed relational |
