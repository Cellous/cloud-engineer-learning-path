# Cloud Definitions

## CIDR
Classless Inter-Domain Routing.
Method for defining subnet ranges using prefix lengths.

## Compute Engine
Google Cloud compute and hosting service for running virtual machines.

## Managed Instance Group (MIG)
A collection of identical VM instances managed from an instance template.

## Cloud Run
Serverless platform for deploying containers.

## Cloud Run Functions
Event-driven serverless functions platform.

## BigQuery
Google Cloud enterprise data warehouse for analytics.

## Cloud SQL
Managed relational database service.

## Spanner
Globally distributed relational database service.
