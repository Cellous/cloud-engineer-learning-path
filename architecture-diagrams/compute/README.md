# Recognition Patterns

For the Associate Cloud Engineer exam:

- Instance Template → Managed Instance Group → Autoscaler
- Template changes require rolling updates
- Startup scripts execute during VM initialization
- Snapshots are incremental after the first full snapshot
- Managed Instance Groups provide self-healing capabilities
- Load balancers distribute traffic across healthy instances
